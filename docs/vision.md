# Vision

SP404 Toolkit will be a trustworthy, open and local-first replacement for the
obsolete Roland SP-404SX Wave Converter. It accepts ordinary WAV files from
any DAW or recorder, helps map and validate them, and will produce a verified
SD-card package for SP-404SX/SP-404A hardware.

The first target is intentionally narrow. Support for other samplers, cloud
accounts, AI tagging and a full audio editor are outside the initial release.

## Principles

- DAW-independent WAV input
- local browser processing and no required account
- no automatic gain or sound changes
- actionable validation instead of automatic mastering
- reusable, browser-independent core logic
- verified compatibility before feature count
- explicit consent for every transformative operation
